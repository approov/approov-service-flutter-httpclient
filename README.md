# Approov Service for Flutter HttpClient

A wrapper for the iOS [Approov SDK](https://github.com/approov/approov-ios-sdk) and Android [Approov SDK](https://github.com/approov/approov-android-sdk) to enable easy integration when using [`Flutter`](https://flutter.dev) for making API calls you want to protect with Approov.

See the [Quickstart](https://github.com/approov/quickstart-flutter-httpclient) for a full integration example.
